# Abbie Integration

TurboKit includes project-level abbie config for loop-style execution.

## .loop.json

```json
{
  "project": {
    "name": "turbokit",
    "team": "TEMP",
    "workspace": "luke-labs"
  },
  "flywheel": "turborepo",
  "commands": {
    "lint": "pnpm lint",
    "typecheck": "pnpm typecheck",
    "build": "pnpm build",
    "test": "pnpm test"
  }
}
```

2. run tests
3. run build
