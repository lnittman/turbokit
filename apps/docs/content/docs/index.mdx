# TurboKit

TurboKit is a Convex-first template for shipping realtime apps quickly.

## What ships by default

- app shell with sidebar + overlay + command palette
- five starter layouts: dashboard, feed, chat, kanban, canvas
- settings overlay for account, appearance, and notifications
- Convex backend package with typed APIs and notification preferences
- docs app (this site) with starter IA
